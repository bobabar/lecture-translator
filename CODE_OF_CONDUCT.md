# Code Of Conduct

This project should be welcoming, practical, and respectful.

Please:

- Be kind and constructive.
- Assume good intent while still being clear about problems.
- Keep discussion focused on improving the software and the learning experience it supports.
- Avoid harassment, personal attacks, or exclusionary language.

Maintainers may remove comments, issues, or pull requests that make the project less safe or less useful for contributors.
