# Security Policy

## Reporting A Vulnerability

Please do not open public issues for security-sensitive reports. Email the maintainer or use a private GitHub security advisory if one is available for the repository.

Useful reports include:

- Affected version or commit
- Reproduction steps
- Expected and actual behavior
- Any relevant logs or crash output

## Runtime Privacy

Lecture Translator is designed to run transcription and translation locally. Microphone audio is processed on the device by the bundled Whisper runtime in release builds.
